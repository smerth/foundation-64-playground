{{#markdown}} # Foundation 6 Javascript Integration Add markup to a page. Install the javascript package. ```javascript yarn add mixitup ``` Import the library. @ app.js ```javascript import mixitup from 'mixitup'; ``` Instantiate the library on an element @ app.js ```javascript var mixedup = mixitup('.mixitup-container'); ``` And that's that! Except that the library loads on every page. {{/markdown}}
